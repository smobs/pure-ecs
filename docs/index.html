A Pure Functional Entity Component System for PureScript

Features

  • Type-safe component composition with row polymorphism
  • Archetype-based storage for cache efficiency
  • Pure functional architecture (no mutable state)
  • Compile-time access tracking for systems
  • Entity lifecycle management with versioning
  • Monadic World API for clean composition
